El proyecto

Cognition es una plataforma de IA aplicada para empresas medianas en Argentina. El producto real existe — el desafío fue construir un sitio que explique una tecnología compleja a una audiencia no técnica, genere confianza enterprise y convierta demos.

El proyecto más complejo del portfolio: 11 páginas inter-relacionadas, calculadora de ROI en JS puro, un chat con intents para calificar leads, animaciones de datos en tiempo real y PWA instalable.

Reto y respuesta

01

El problema

Una demo existente del producto en React con bundle de 4MB que tardaba 8s en cargar en mobile. Sin SEO, sin Schema.org, sin conversión medida. El equipo de ventas perdía leads porque el sitio no transmitía seriedad enterprise antes de la primera reunión.

02

La solución

Reescritura completa en vanilla JS. Arquitectura multi-página con routing custom. ROI calculator que persiste en sessionStorage. Chat con 12 intents mapeados para calificar leads antes del formulario. Service Worker con cache estratégico por tipo de recurso.

Decisiones técnicas

⬡

ROI Calculator

Calculadora interactiva en JS puro con inputs deslizantes y resultado animado. Persiste en sessionStorage para que el usuario no pierda sus datos al navegar entre páginas. El resultado se pre-llena en el formulario de demo.

⬡

Chat con intents

Chat widget sin dependencias externas. 12 intents mapeados (pricing, integrations, security, demo, etc.). Respuestas en árbol de decisión. Cuando el usuario pide una demo, el formulario se pre-llena con el contexto de la conversación.

⬡

Service Worker

Precaching del shell en install. Network-first para páginas HTML. Cache-first para assets. Versioning por hash para invalidar cache en deploy. Funcionamiento offline con fallback de error custom.

⬡

Animaciones de datos

Canvas API para animaciones de red neuronal en hero. requestAnimationFrame con throttle a 30fps. Pausa automática con IntersectionObserver cuando el canvas sale del viewport. Fallback SVG estático para prefers-reduced-motion.

⬡

Schema.org SaaS

SoftwareApplication con applicationCategory: "BusinessApplication". Offer para pricing. FAQPage en la página de preguntas. BreadcrumbList en todas las páginas secundarias. Rich results en Google validados.

⬡

Performance 11 páginas

CSS compartido en un solo archivo con lazy-loading de secciones. JS modularizado por feature con dynamic import. Total bundle: 38KB JS + 24KB CSS comprimidos. LCP medio de 2.1s en 3G lento según field data.
