Política de Privacidad

Última actualización: mayo de 2026

Este sitio web (rony-cozzi.netlify.app) es operado por Rony Cozzi, desarrollador web independiente con base en Córdoba, Argentina. Esta política describe qué datos se recopilan, cómo se usan y cuáles son tus derechos, en cumplimiento con la Ley 25.326 de Protección de Datos Personales (Argentina) y principios del GDPR.

1. Datos que recopilamos

Este sitio web recopila datos en dos situaciones:

  • Formulario de contacto: cuando enviás un mensaje, se recopila tu nombre, dirección de email y el texto del mensaje. Estos datos se almacenan en Netlify Forms y se reenvían a ronycozzi5@gmail.com.
  • Analytics de privacidad: este sitio puede usar Plausible Analytics (sin cookies, sin fingerprinting, sin datos personales). Los datos son anónimos y agregados.

Este sitio no usa cookies de seguimiento, Google Analytics, Facebook Pixel ni ninguna forma de rastreo individual de usuarios.

2. Cómo usamos los datos

  • Los datos del formulario de contacto se usan exclusivamente para responder a tu consulta.
  • No se comparten con terceros salvo los proveedores de servicio (Netlify, Gmail) necesarios para procesar tu mensaje.
  • No se usan para marketing ni se venden bajo ninguna circunstancia.

3. Retención de datos

Los mensajes del formulario se retienen mientras sean relevantes para la comunicación activa, y se eliminan a pedido. No existe base de datos propia — los datos residen en la infraestructura de Netlify y Google Gmail, ambos con sus propias políticas de retención.

4. Tus derechos

Bajo la Ley 25.326 y el GDPR tenés derecho a:

  • Acceder a los datos personales que hayás proporcionado
  • Solicitar la corrección de datos incorrectos
  • Solicitar la eliminación de tus datos
  • Oponerte al tratamiento de tus datos

Para ejercer cualquiera de estos derechos, escribí a ronycozzi5@gmail.com. Respondo en menos de 72 horas hábiles.

5. Seguridad

El sitio está alojado en Netlify con HTTPS (TLS 1.3), headers de seguridad (CSP, X-Frame-Options, HSTS) y sin bases de datos propias accesibles externamente. No podemos garantizar seguridad absoluta — ningún sistema en internet puede — pero aplicamos las prácticas estándar de la industria.

6. Cambios a esta política

Si esta política cambia de forma significativa, se actualizará la fecha en el encabezado. Para cambios menores no se enviarán notificaciones.
